# NestJS Backend Internship Challenge

This repository is planned as a Project and Task Management REST API. NestJS
and TypeScript are planned for the application, with PostgreSQL persistence,
JWT authentication, testing, and API documentation. The project is currently
in the planning stage; application features have not yet been implemented.
